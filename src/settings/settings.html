QA Annotator — Profiles

Profiles describe a project's URL patterns, framework, locale strategy, and Jira / Figma defaults. Generate one with the /qa:init Claude Code plugin and import it here.

GitHub repository · Claude Code plugin setup · Extension install guide

Pick the workflow that matches how you'll mainly use this extension. Each mode toggles the right capture sources and modal panels by default — you can fine-tune below.

Theme color

    
One color drives the whole extension: inspector outline, multi-pick rings, hover tooltip, captured-screenshot border, AND the popup / modal / settings UI accents. Useful when the default conflicts with the site's own UI, or for color-coding QA conventions across projects.
